# CodeAlpha_SalesPrediction
Sales forecasting project using advertising spend data (TV, Radio, Newspaper) to predict revenue. Compares 6 regression models including Polynomial and Gradient Boosting. Delivers actionable insights on the highest-impact advertising channels. Built with Python and Scikit-learn.
● Predict future sales based on factors like advertising spend, target segment and platform. ● Prepare data through cleaning, transformation and feature selection. 
● Use regression or time series models to forecast sales. 
● Analyze how changes in advertising impact sales outcomes. 
● Deliver actionable insights for business marketing strategies.
